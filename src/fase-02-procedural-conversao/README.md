# Fase 2 — Conversor Procedural (Conversão de Moedas)

## Objetivo
Implementar uma versão procedural que suporte:
- BRL -> Estrangeira
- Estrangeira -> BRL
- Estrangeira -> Estrangeira (via BRL)
- Políticas: FIXA, PERSONALIZADA, COMERCIAL, TURISMO

## Como executar
dotnet run dentro da pasta do projeto.
