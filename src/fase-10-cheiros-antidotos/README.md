# Fase 10 — Cheiros e Antídotos (Refatorações com Diffs Pequenos)

Esta fase aplica melhorias arquiteturais *cirúrgicas* ao projeto, com mudanças muito pequenas,
guiadas por princípios SOLID, evitando “big refactors”.

Cada cheiro inclui:
1. Descrição no contexto do projeto
2. Código ANTES
3. Código DEPOIS
4. Antídoto aplicado
5. Princípio SOLID envolvido
6. Teste que prova segurança

Todos os cheiros estão documentados em  
**CheirosEAntidotos.md**

### Objetivo da Fase
- Melhorar a qualidade do código passo a passo.
- Aplicar refatorações pequenas, seguras e guiadas por testes.
- Demonstrar domínio sobre ISP, DIP, SRP, OCP.
- Criar um repositório profissional e evolutivo.

### Como navegar
- `CheirosEAntidotos.md` → refatorações detalhadas (antes/depois).
- Pasta `tests/fase-10-tests/` → testes que comprovam segurança.
