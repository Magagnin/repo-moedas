# Fase 11 — Mini-projeto (CurrencyRate)

Este projeto demonstra:
- Contratos segregados (IReadRepository / IWriteRepository)
- Repositórios InMemory e JSON (persistência)
- Serviço que depende apenas de contratos (CurrencyRateService)
- CLI demo simples (Program.cs)

Como executar demo:
cd src/fase-11-mini-projeto
dotnet run


Como rodar testes:


cd tests/fase-11-tests
dotnet test
